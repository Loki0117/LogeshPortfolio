 Logesh Portfolio

A modern React-based developer portfolio built to showcase my projects, technical skills, and learning journey as a software developer.

The goal of this portfolio is to present my work in a clean and structured way so recruiters, developers, and collaborators can easily explore what I’ve been building.

 Live Website
https://logesh-portfolio-beta.vercel.app/

 About This Project

This portfolio was created to bring together my projects, skills, and development journey in one place.

While learning and building projects, I realized the importance of having a central space to showcase work and progress. This portfolio reflects the technologies I work with and the projects I’ve developed while growing as a developer.

The design focuses on simplicity, readability, and responsiveness across different devices.

 Features

• Hero Section – Quick introduction and overview
• About Section – A brief summary of my background and interests
• Projects Section – Showcases projects with descriptions and links
• Skills Section – Technologies and tools I work with
• Contact Section – Easy ways to connect with me
• Responsive Design – Works across desktop, tablet, and mobile devices

 Getting Started
Clone the repository

git clone https://github.com/Loki0117/LogeshPortfolio.git

Navigate to the project directory
cd LogeshPortfolio

Install dependencies
npm install
Run the development server
npm start

The application will run locally at:

http://localhost:3000
